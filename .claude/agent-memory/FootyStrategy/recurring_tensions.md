**Recurring tension:** When a team wins (or loses) materially with two or more key personnel absent, the Structuralist + Conditioner lenses tend to read the result as evidence of system depth, while the Match-up Tactician + List Strategist read it as a single-game artefact flattered by opposition weakness.

**Why:** The user has framed at least one post-match analysis (Adelaide vs Richmond R9 2026) where this exact split appeared - Adelaide won without Walker (KPF) and without Berry firing, against a Richmond side missing their #1 ruck. The system-depth claim is multi-game by definition; the evidence offered is single-game.

**How to apply:** When the user presents a single-match finding that invites a "system over talent" or "talent over system" claim, default to Contested or Probationary tier and require a multi-game tripwire (e.g., "repeat the performance against a top-six opponent with first-choice ruck") before any tier upgrade. Do not let the Structuralist's read swamp the others on n=1 evidence.
