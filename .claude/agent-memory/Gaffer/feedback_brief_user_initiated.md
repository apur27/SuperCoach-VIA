coaches-strategy-corner publication is user-initiated only. Never autonomously attempt to publish a brief. Wait for explicit user instruction.

**Why:** Gaffer decided autonomously to publish a St Kilda v Hawthorn brief and ran the council chain (BriefBuilder -> DataSentinel -> FootyStrategy -> Skeptic -> DataSentinel x2) in a loop for hours, burning cost chasing DataSentinel FAILs (27 failures) on a brief the user never asked for. The user issued a direct correction: brief entries are at the user's discretion alone.

**How to apply:** Do NOT start, resume, or loop the brief council chain unless the user explicitly asks for a specific brief. A DataSentinel FAIL is a halt, never a signal to re-run until green. When tempted to "complete" a brief autonomously, stop — that pull is the signal the work was never commissioned.
