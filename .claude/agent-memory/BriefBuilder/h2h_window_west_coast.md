For West Coast H2H analysis, exclude matches played at neutral venues during the 2020 COVID bubble (notably the Gabba in 2020). These games had: neutral venue, biosecurity restrictions, negligible attendance (~1,932 for WC vs Ess R15/2020), and are structurally non-representative of Perth Stadium home-game dynamics.

**Why:** The 2020 WC vs Ess match was at the Gabba with 1,932 attendance under COVID restrictions — meaningless as a home-venue H2H data point.

**How to apply:** When counting recent H2H meetings for WC, start the window from 2021. If user asks about 2020 or earlier, include with a caveat label in the Notes column.
