Two files match `kennedy_matthew_*_performance_details.csv`:
- `kennedy_matthew_04021970_performance_details.csv` — no Western Bulldogs 2026 data (confirmed by grep returning no results)
- `kennedy_matthew_06041997_performance_details.csv` — active Western Bulldogs player in 2026; use this one

**How to apply:** When building any brief that includes Kennedy Matthew (WB), always glob both files and confirm which has current-year WB data before proceeding. The 1970 DOB file is a historical player (career predates 2026).

See also: [[player-disambiguation-roberts]] for the same pattern with Roberts Archie.
