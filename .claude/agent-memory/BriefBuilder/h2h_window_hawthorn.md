Use last 8 meetings when assembling H2H for Hawthorn matchups in 2024–2026 briefs.

**Why:** Hawthorn's dynasty era (pre-2017) and rebuild nadir (2018–2020) represent structurally different list compositions. Post-2021 data captures the current roster identity. Applied in St Kilda vs Hawthorn R13 2026 brief without pushback.

**How to apply:** For any Hawthorn H2H ledger, default window = last 8 meetings from current season back. If those 8 meetings include pre-2021 games, note the era gap explicitly and consider whether the old data adds signal or noise.
